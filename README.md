# PCOS Prediction System using Machine Learning

## Project Overview
Polycystic Ovary Syndrome (PCOS) is a common hormonal disorder affecting women of reproductive age. Early detection can help in better medical management and lifestyle changes.

This project uses a **Machine Learning model** to predict the likelihood of PCOS based on various health parameters. The system provides a **web-based interface** where users can enter medical information and receive a prediction result.

The application is built using **Python, Flask, and Machine Learning techniques**.

---

## Features

- User-friendly web interface
- Input health parameters related to PCOS
- Machine learning-based prediction
- Instant result generation
- Simple and interactive design
- Helps in early awareness of PCOS risk

---

## Technologies Used

- Python
- Flask
- Machine Learning
- Scikit-learn
- HTML
- CSS
- JavaScript
- Pandas
- NumPy

## How the System Works

1. User enters health parameters through the web interface.
2. The backend processes the input data.
3. The trained machine learning model analyzes the data.
4. The system predicts whether the user has a risk of PCOS.
5. The prediction result is displayed on the screen.
   

## How to Run the Project

1. Clone the repository

git clone https://github.com/yourusername/pcos-prediction.git

2. Navigate to the project folder

cd pcos-prediction

3. Install required libraries

pip install -r requirements.txt

4. Run the application

python app.py

5. Open the browser and go to

http://127.0.0.1:5000
